## 2. 产品结构

页面流程：

```text
首页
→ 24 题答题页
→ 生成报告加载页
→ 结果报告页
→ 复制摘要 / 重新诊断
```

题库结构：

```text
工作系统：6 题
能量系统：6 题
成长系统：6 题
生活系统：6 题
```

算分逻辑：

```text
A = 1 分
B = 2 分
C = 3 分
D = 4 分

最高分系统 = 主结果
第二高分系统 = 次级问题
总分 = 风险等级
```

结果类型：

```text
职业错位型
能量耗尽型
成长停滞型
活法失衡型
```
